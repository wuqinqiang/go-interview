    grpc-shop 介绍 #

    项目介绍 #

    grpc-shop 是一个基于 grpc 开发的，简单的微服务电商项目。分为三部分:

    product-srv #

    商品服务。github

    order-src #

    订单服务。github

    user-srv #

    用户服务。github

    读者定位 #

    • 了解基本的 Go 语法，能写点代码。
    • 了解或者刚开始接触 grpc，官方的 demo 已经无法满足，想开发一些简单的应用。
    • 想了解 go 生态中的一些工具或者包。
